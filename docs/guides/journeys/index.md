# Journeys

Step-by-step guides for common heliosApp workflows.

## Available Journeys

| Journey | Description | Time |
|---------|-------------|------|
| [Setting Up heliosApp Development](./setting-up-heliosapp-development.md) | Set up your local development environment | 45-60 min |
| [Adding a New Feature Component](./adding-a-new-feature-component.md) | Add a new component to the SolidJS renderer | 30-45 min |
| [Deploying to Production](./deploying-to-production.md) | Deploy heliosApp to Kubernetes production | 60-90 min |

## Prerequisites

All journeys assume you have:
- Completed the previous journey in sequence
- Access to heliosApp repository
- Bun 1.2.20+ installed
- Basic familiarity with TypeScript and SolidJS

## Quick Navigation

```bash
# Start with setup
open ./docs/guides/journeys/setting-up-heliosapp-development.md

# Then add a feature
open ./docs/guides/journeys/adding-a-new-feature-component.md

# Finally, deploy
open ./docs/guides/journeys/deploying-to-production.md
```
